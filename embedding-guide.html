Troubleshooting

Tool Not Loading?

  • Check that your site is using HTTPS (not HTTP)
  • Verify the URL is correct
  • Disable security extensions temporarily
  • Clear browser cache
  • Contact us at support@metaphordefinition.com

Mobile Not Working?

  • Use width="100%" for responsive design
  • Test on actual mobile device
  • Adjust height for mobile (may need less height)
  • Check for browser compatibility

Performance Issues?

  • Lazy load the iframe (load on scroll)
  • Don't embed multiple tools on same page
  • Use smaller height if possible
  • Monitor with PageSpeed Insights
